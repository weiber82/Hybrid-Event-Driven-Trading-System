Stock Price Prediction Method Based on the XGBoost Algorithm
1. 研究背景與動機

金融時間序列具有非線性、噪聲高與週期性不明顯等特性，使傳統線性模型難以有效預測市場波動。隨著機器學習在預測任務中的成功案例增加，研究者開始探索使用 XGBoost 等集成方法，以改善市場預測的準確度與穩定性。

2. 研究目的

本研究旨在評估 XGBoost 在股價預測任務中的可行性，並比較其與傳統模型之間的預測效能差異。同時探討特徵工程、資料處理與重要特徵的模型貢獻。

3. 研究方法與架構

研究流程如下：

3.1 資料前處理與特徵工程

使用多種技術指標（如 RSI、MACD、KDJ、移動平均線）

生成滯後特徵（lag features）

標準化處理價格與成交量

3.2 模型訓練

採用 XGBoost 作為主模型

與其他方法比較（如 Backpropagation Neural Network）

以 RMSE、MAE 進行模型評估

3.3 特徵重要度分析

使用樹模型的 split gain 與 weight 進行特徵排序，辨識對預測影響最強的技術指標與滯後變數。

4. 數據來源與實作

使用中國 A 股每日股價資料

訓練與測試集按時間順序切割

設置多組特徵組合進行效果比較

5. 研究結果

XGBoost 的預測表現優於傳統神經網路與線性模型

技術指標與滯後特徵對模型預測貢獻高

模型能有效捕捉部分價格變動方向

特徵重要度分析可協助理解市場訊號來源

6. 研究貢獻

展示 XGBoost 在非線性金融預測問題中的優勢

強調特徵工程在機器學習模型中的重要性

提供預測模型的特徵貢獻分析方式

為 XGBoost 在金融市場的應用建立實證基礎

7. 研究限制與未來方向

限制：

採用日資料，無法反映高頻市場微結構

標籤設計偏向回歸預測，缺乏多分類事件視角

垂直資料集仍侷限於單一市場

未來方向：

擴展至更高頻率資料

引入多市場或跨資產特徵

結合深度學習或混合模型架構

8. 總結

本研究證明 XGBoost 在金融時間序列預測任務中具有明顯優勢，能處理多維度特徵並提供模型可解釋性。研究指出技術指標與滯後特徵在預測中仍具有重要作用，並凸顯集成樹模型在擷取非線性市場訊號方面的強度。

9. Applicability to This System（可用於本系統的參考重點）
9.1 XGBoost 適用於金融非線性訊號處理

研究展示 XGBoost 可有效處理金融資料的噪聲與變異性，對於事件偵測類任務（如 Breakout、Reversal）具有高度參考價值。

9.2 技術指標與滯後特徵的效用

本研究支持在金融預測任務中加入多種技術指標與滯後特徵；此結果可作為高頻特徵工程（如 rolling features、microstructure features）的基礎。

9.3 特徵重要度可作為語義層輸出依據

研究使用 XGBoost feature importance 評估特徵貢獻，對後續 L1（SHAP-to-text 語義轉譯）提供基礎。

9.4 支援事件驅動架構的第一階段分類模型

雖然研究目標為回歸預測，但其模型架構與特徵工程方式適用於本系統 L0 的事件分類任務（如 Triple-Barrier 多分類）。

9.5 可作為 XGBoost 金融應用的文獻依據

本研究補強集成樹模型在市場預測中的使用案例，可用於章節中「L0 模型選用理由」的文獻支持。
