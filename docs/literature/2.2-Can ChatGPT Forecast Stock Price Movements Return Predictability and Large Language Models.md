Can ChatGPT Forecast Stock Price Movements? Return Predictability and Large Language Models

1. 研究背景與動機

金融市場長期研究資訊如何被投資人吸收與反應。然而，要同時衡量「事件真正的經濟意涵」以及「市場對事件的即時反應」，在實證中非常困難。

本研究提出一個新的工具：
使用大型語言模型（LLM）作為“經濟意涵解讀器”，檢驗市場對新聞事件的反應是否到位。

2. 研究目的

評估 LLM（ChatGPT-4）能否從新聞標題判斷事件的經濟方向

測試 LLM 是否能預測：

初始市場反應（initial reaction）

隨後的價格漂移（post-announcement drift）

比較不同 LLM（GPT-1, GPT-2, BERT, Llama2）的能力

建立理論模型解釋：為何 LLM 的資訊處理能力會產生可預測性

3. 研究方法與架構
3.1 Prompt 設計

研究使用固定格式 prompt：

Pretend you are a financial expert.
Answer “YES” if good news, “NO” if bad news, or “UNKNOWN” if uncertain.
Then give a short one-sentence explanation.


（來源：）

將回答轉換成：

YES → +1

UNKNOWN → 0

NO → -1

稱為 GPT-4 score。

3.2 初始反應預測（Immediate reaction）

根據研究結果：

GPT-4 對「新聞發布後立刻的市場反應」預測能力極高

長空組合的 hit rate 約 90%（非可交易，因為反應已經發生）

3.3 價格漂移（Post-announcement drift）

核心發現：

在初始反應後的 1–2 天，股價持續往 GPT-4 的方向漂移

代表市場存在「資訊反應不足（underreaction）」

尤其是：

負面新聞（NO）漂移最強

小型股漂移更明顯

3.4 交易策略與報酬

研究構建基於 GPT-4 score 的每日 long-short 策略：

在不含交易成本下，累積報酬 700%（2021–2024）

交易成本考量：

5 bps 下仍可賺超過 300%

10 bps 仍維持正報酬

大於 20 bps 顯著減弱

3.5 理論模型（LLM adoption & market efficiency）

研究建立一個金融動態模型，證明：

LLM 越大型，其資訊處理能力越接近「專業投資人」

LLM 能改善市場效率

存在一個 “最小模型規模 k*”，只有大型 LLM 才能提供可交易訊號

4. 實證結果

LLM 能準確解讀新聞的經濟意涵

LLM 對短期股票報酬具有顯著可預測性

漂移主要來自負面新聞及小型股

GPT-4 的成效優於：

傳統情緒分數（如 RavenPack）

BERT、GPT-1、GPT-2、Llama2

（GPT-4 能取代傳統 sentiment 的預測力）

5. 研究貢獻

首次展示一般用途 LLM（非金融訓練）能精準判斷財經新聞的經濟意涵

提供量化市場資訊反應的全新工具：LLM score

以事件為單位，建構可解釋的市場效率衡量方式

提供理論模型支持市場因 LLM 採用而變得更有效率

6. 限制與未來方向
限制

基於美股新聞資料，其他市場未驗證

使用 headline，未處理更複雜文本

模型推論成本較高

未來方向

建構 domain-specific LLM for finance

處理更複雜的財報、會議記錄、法說會內容

更完整的多因子市場資訊處理分析

7. 與本系統的適用性（L2 層）
7.1 完全對應 L2：「LLM 作為事件解讀與決策代理」

本研究驗證：
LLM 能以自然語言推理方式，準確判斷事件的經濟方向。

這正是 L2（Decision Agent）的核心：

接收 L1 的語義摘要

判斷是否採用訊號

輸出 BUY / SELL / HOLD

並提供自然語言理由

7.2 支持 L2 的「語義推理」與「meta-verification」概念

研究中 ChatGPT 的操作流程：

輸入一段簡短新聞

LLM 給出方向性判斷

再給出理由解釋

這與本系統 L2 設計完全一致。

7.3 支持“市場存在 underreaction，因此 LLM 訊號有價值”

該研究指出：

市場對負面新聞反應不足

小型股存在顯著漂移機會

這可直接用於 L2 風險調整與策略強化。

7.4 可作為 L2 層最重要的核心引用之一

此研究能提供：

為何使用 LLM 來解讀盤中事件

為何 LLM 的語義推論具經濟價值

為何將 LLM 作為二階判斷（meta reasoning）是合理的
